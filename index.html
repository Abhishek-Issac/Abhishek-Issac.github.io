My Story

Developer, Builder, Tech Enthusiast

I'm Abhishek Issac, a passionate developer based in Kerala, India. My journey into programming started with curiosity about how things work behind the scenes — and that curiosity has evolved into a deep passion for building software that solves real problems.

My primary expertise lies in Python and Telegram Bot development using frameworks like Pyrogram and Telethon. I've built and deployed multiple bots that handle thousands of users, from series streaming platforms to group management tools and automated forwarding systems.

Beyond bots, I work with MongoDB, PostgreSQL, Redis, and Docker, managing production workloads on DigitalOcean Ubuntu VPS servers. I'm constantly exploring new technologies — from Android app development with Kotlin to Go-based Telegram libraries, web scraping, and API integrations.

When I'm not coding, you'll find me exploring the latest in mobile tech, watching anime and series, gaming, or tinkering with phone hardware and custom ROMs. I believe in learning by doing and sharing knowledge with the community.
